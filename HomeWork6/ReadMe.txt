В пакете runners:

в файле EasySearchMain подсчет слов только с помощью реализации интерфейса без использования регулярных выражений. При этом с поиск с учетом регистра похож на правду, а вот без учета регистра хромает. Пробовала использовать String stopChars, как в классной работе - ничего не изменилось.

в файле SearchWithDecorator реализован поиск с помощью декораторов удаления знаков пунктуации и перевода всего текста в нижний регистр. Слово мир не совпадает на 1 с исходником.
